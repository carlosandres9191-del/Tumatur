Tuma Tour Web

Archivos principales:
- index.html -> página principal enfocada al cliente
- crm.html -> panel interno conectado por localStorage
- styles.css -> estilos y animaciones
- app.js -> lógica de reservas y CRM
- assets/ -> imágenes y logo

Cómo usar:
1. Abre index.html en tu navegador.
2. Para revisar las reservas, abre crm.html en el mismo navegador.
3. La conexión actual funciona en el mismo dispositivo/navegador.
4. Para uso real online con múltiples usuarios, se necesita backend y base de datos.
